Concentration of the Empirical Median around the mean

\( \renewcommand{\P}{\mathbb{P}} \newcommand{\E}{\mathbb{E}} \newcommand{\R}{\mathbb{R}} \newcommand{\1}{\textbf{1}} \newcommand{\Med}{\mathrm{Med}} \newcommand{\N}{\mathbb{N}} \)

Theorem

Let \(X\) be a real random variable with law \(P\) and finite variance \(\sigma^2>0\).
Let \(n \in \N\) with \(n\ge 6\), let \(X_1,\dots,X_n\) be an i.i.d sample from \(P\).
Denote \(\Med(X_1^n)\) an empirical median of \(X_1,\dots,X_n\) and \(\Med(P)\) a theoretical median of \(P\).
Then, for any \(\lambda >0\), we have for any $ 0 < t ≤ \frac{8}{81n}$, \[\P\left(|\Med(X_1^n)-\E[X]| \le \sigma + 18\sigma\sqrt{\frac{t}{2n}} \right) \ge 1- 2\exp\left(-\frac{t}{\left(1+18\sqrt{\frac{t}{2n}}+81\frac{t}{n}\right)^2 }\right) \]

Remark: The error term \(\sigma\) correspond to a (tight in general) bound on \(|\E[X]-\Med(P)|\). If additionally to the assumptions of the theorem, we suppose that \(P\) is symmetric so that \(\E[X]=\Med(P)\), then one can show that for any $ 0 < t ≤ \frac{8}{81n}$, \[\P\left(|\Med(X_1^n)-\E[X]| \le 18\sigma\sqrt{\frac{t}{2n}} \right) \ge 1- 2\exp\left(-\frac{t}{\left(1+18\sqrt{\frac{t}{2n}}+81\frac{t}{n}\right)^2 }\right).\]

Notations

\(\R\) denotes the set of real numbers, \(\N\) the set of non-negative integers. For a random variable \(X\), \(\E[X]\) denotes its expectation and \(\P(X \in A)\) the probability that \(X\) belongs to a set \(A\).

Proof of the theorem

1. For any \(\lambda >0\), we have

\begin{equation}\label{eq:concentration_to_count} \P\left(\Med(X_1^n) \ge \E[X]+\sigma+ \lambda \right) \le \P\left(\sum_{i=1}^n \1\{X_i \ge \E[X]+\sigma+\lambda\} \ge \lfloor n/2 \rfloor \right). \end{equation}

1.1. For any \(a\in \R\), if \(\Med(X_1^n)\ge a\) then \(\sum_{i=1}^n \1\{X_i \ge a\}\ge \lfloor n/2\rfloor\)

1.1.1. Proof of 1.1

Let \(X_{(i)}\) denote the sorted sample: \(X_{(1)}\le X_{(2)}\le \dots\le X_{(n)}\), by definition, \(\Med(X_1^n) \le X_{(\lceil n/2\rceil)}\), hence the result.

1.1.1.1. Among \(X_1,\dots,X_n\) there are at least \(\lfloor n/2\rfloor\) elements larger than \(a\).
1.1.1.1.1. Proof of 1.1.1.1

We took the definition that \(\Med(X_1^n)\) is any element in the interval \([X_{(\lfloor n/2\rfloor)}, X_{(\lceil n/2\rceil)}]\). Hence, for ay \(j \ge \lceil n/2\rceil\), we have \[X_{(j)}\ge X_{(\lceil n/2\rceil)}\ge \Med(X_1^n).\] Which implies that there are at least \(n - \lceil n/2\rceil = \lfloor n/2\rfloor\) elements among \(X_1,\dots,X_n\) that are larger than \(\Med(X_1^n)\).

1.2. Conclude that Equation \eqref{eq:concentration_to_count} is true.

1.2.1. Proof of 1.2

Apply Step 1.1 to \(a=\Med(P)+\lambda\).

2. For any \(\lambda> 4\sigma/n\), we have

\begin{equation}\label{eq:hoeffding} \P\left(\Med(X_1^n) \ge \E[X]+\sigma + \lambda \right) \le \exp\left(-2n\left(\frac{\lambda \sigma }{9(2\sigma^2+2\lambda\sigma+\lambda^2) }\right)^2\right). \end{equation}

2.1. Let \(p(\lambda)=\P(X_1 \ge \E[X]+\sigma+\lambda)\), from Hoeffding's inequality, for any \(\lambda>0\) such that \(p(\lambda) \le \frac{1}{n}\lfloor n/2\rfloor\), we have

\[\P\left(\Med(X_1^n) \ge \E[X]+\sigma+ \lambda \right) \le \exp\left(-2n\left(\frac{1}{n}\lfloor n/2 \rfloor-p(\lambda)\right)^2\right).\]

2.1.1. Proof of 2.1

From Step 1, it is sufficient to control \(Z=\sum_{i=1}^n \1\{X_i \ge \Med(P)+\lambda\}\). \(Z\) is a sum of i.i.d random variables bounded by \(1\) and with mean \(p(\lambda)\). Hoeffding's inequality can be applied to \(Z\) as long as \(p(\lambda) \le \frac{1}{n}\lfloor n/2 \rfloor\).

2.2. For any \(\lambda > 0\), we have $p(λ) ≤ \frac{\sigma^2}{2\sigma^2+2\lambda \sigma+\lambda^2 }.$

2.2.1. Proof of 2.2

From Cantelli's inequality, for any $λ >0 $ we have \[\P(X_1 \ge \E[X]+\sigma+\lambda) \le \frac{\sigma^2}{\sigma^2+(\lambda+\sigma)^2 }.\]

2.3. If \(\lambda > 4\sigma/n\), then $ \frac{1}{n}⌊ n/2⌋-p(λ)≥ \frac{\lambda \sigma }{9(2\sigma^2+2\lambda\sigma+\lambda^2) }≥ 0.$

2.3.1. Proof of 2.3

For any \(n \ge 6\) and \(\lambda > 0\), from Step 2.2, \[ \frac{1}{n}\lfloor n/2\rfloor-p(\lambda)\ge \frac{1}{2}-\frac{1}{n}-\frac{\sigma^2}{2\sigma^2+2\lambda\sigma+\lambda^2 }.\] This prove one part of the inequality. Then, using that \(n \ge 6\),

\begin{align*} \frac{1}{2}-\frac{1}{n}-\frac{\sigma^2}{2\sigma^2-2\lambda\sigma+\lambda^2 }&= \frac{n-2}{2n}-\frac{\sigma^2}{2\sigma^2+2\lambda\sigma+\lambda^2 }\\ &= \frac{(n-2)(2\sigma^2+2\lambda\sigma+\lambda^2)-2n\sigma^2}{2n(2\sigma^2+2\lambda\sigma+\lambda^2) }\\ &=\frac{-4\sigma^2+2(n-2)\lambda\sigma+(n-2)\lambda^2}{2n(2\sigma^2+2\lambda\sigma+\lambda^2) } \end{align*}

Then, using that \(n-2 \ge 2n/3\) because \(n \ge 6\),

\begin{align*} \frac{1}{2}-\frac{1}{n}-\frac{\sigma^2}{2\sigma^2-2\lambda\sigma+\lambda^2 }&\ge \frac{-4\sigma^2+4\frac{n}{3}\lambda\sigma+2\frac{n}{3}\lambda^2}{2n(2\sigma^2+2\lambda\sigma+\lambda^2) } \end{align*}

Now, suppose that \(\lambda \ge 4\sigma/n\), then \(n\lambda\sigma\ge 4\sigma^2\) and then,

\begin{align*} \frac{1}{2}-\frac{1}{n}-\frac{\sigma^2}{2\sigma^2-2\lambda\sigma+\lambda^2 }&\ge \frac{\frac{n}{3}\lambda \sigma + \frac{2n}{3}\lambda^2}{3n(2\sigma^2+2\lambda\sigma+\lambda^2) }\\ &\ge \frac{\lambda \sigma }{9(2\sigma^2+2\lambda\sigma+\lambda^2) } \end{align*}

2.4. Proof of Step 2

Inject the inequality from Step 2.3 in Step 2.1 to get the result.

3. From the previous steps with \(\lambda = 18\sigma\sqrt{\frac{t}{2n}}\), using some elementary algebra we conclude that the theorem is true.

3.1. Proof of 3

Take \(\lambda = 18\sigma\sqrt{\frac{t}{2n}}\). First, we verify that \(\lambda > 4\sigma/n\). Indeed, having \(t \le 8/(81n)\), we have \[\lambda \ge 18\sigma\sqrt{\frac{4}{81n^2}}=4\sigma/n \] Hence, we can apply Step 2 and conclude that

\begin{align*} \P\left(\Med(X_1^n) \ge \E[X]+\sigma + 18\sigma\sqrt{\frac{t}{2n}} \right) &\le \exp\left(-t\frac{4\sigma^4}{(2\sigma^2+2\lambda\sigma+\lambda^2)^2 }\right)\\ &\le \exp\left(-\frac{t}{\left(1+18\sqrt{\frac{t}{2n}}+81\frac{t}{n}\right)^2 }\right). \end{align*}

This proves one direction of the theorem. Replace \(X\) by \(-X\) and with the same reasoning

\begin{align*} \P\left(\Med(-X_1^n) \ge \E[-X]+\sigma + 18\sigma\sqrt{\frac{t}{2n}} \right) \le \exp\left(-\frac{t}{\left(1+18\sqrt{\frac{t}{2n}}+81\frac{t}{n}\right)^2 }\right). \end{align*}

Hence the result, by union bound.
